# Python-SSH-BruteForce
一款基于 Python 实现的多线程 SSH 暴力破解工具，支持自定义密码位数自动生成字典，通过多线程并发尝试远程服务器 SSH 账号密码，快速完成口令爆破测试，适用于网络安全学习、口令强度测试场景。

## 项目特点
- 自动生成指定长度的数字密码字典，无需手动准备
- 多线程并发爆破，可自定义线程数量（1-50区间智能限制）
- 集成 paramiko 库，稳定实现 SSH 协议连接与登录验证
- 实时显示爆破进度，直观查看尝试次数与耗时
- 完善的异常处理，规避连接失败、超时等问题，提升爆破稳定性
- 支持自定义 SSH 端口，适配不同服务器配置

## 技术栈
- Python 3.x
- paramiko：实现 SSH 协议连接与登录验证
- 多线程（threading）：提升并发爆破效率
- 队列（Queue）：管理密码尝试任务，保证线程安全
- itertools：生成密码组合，构建密码字典
- sys：解析命令行参数，实现灵活调用

## 安装依赖
需安装 paramiko 库，执行以下命令：
```bash
pip install paramiko
```

## 使用方法
### 命令行调用格式
```bash
python faster_ssh_brute_force.py <目标IP> <用户名> <密码长度> [端口] [线程数]
```

### 示例
1. 基础使用（默认端口22，默认10线程，生成3位数字密码字典）
```bash
python faster_ssh_brute_force.py 192.168.1.100 Administrator 3
```

2. 自定义端口与线程数（端口2222，20线程，生成4位数字密码字典）
```bash
python faster_ssh_brute_force.py 192.168.1.100 root 4 2222 20
```

### 参数说明
| 参数位置 | 参数含义 | 必选/可选 | 说明 |
|----------|----------|-----------|------|
| 1        | 目标IP   | 必选      | 需爆破的SSH服务器IP地址 |
| 2        | 用户名   | 必选      | SSH登录用户名（如root、Administrator） |
| 3        | 密码长度 | 必选      | 生成密码的位数（正整数） |
| 4        | 端口     | 可选      | SSH服务端口，默认22 |
| 5        | 线程数   | 可选      | 并发线程数，默认10，自动限制在1-50之间 |

## 核心功能实现
1. 密码字典生成：通过 itertools.product 生成指定长度、指定字符集（默认数字0-9）的密码组合，保存至本地文件
2. 多线程爆破：采用线程池+队列模式，多线程并发尝试密码登录，提升爆破效率
3. SSH登录验证：利用 paramiko 库建立SSH连接，捕获认证异常，判断密码是否正确
4. 进度显示：实时计算并显示爆破进度、已尝试次数，直观反馈任务状态
5. 异常处理：针对连接失败、超时、认证错误等异常做单独处理，避免程序崩溃

## 注意事项
1. 本工具仅用于 **网络安全学习、口令强度测试、授权范围内的服务器安全自查**，请勿用于未授权的远程服务器爆破，否则后果自负。
2. 线程数建议控制在10-20之间，过多线程可能导致被目标服务器屏蔽IP。
3. 密码字典默认仅生成数字组合，如需扩展字符集（如字母、符号），可修改 `generate_password_dict` 函数中的 `chars` 参数。
4. 运行前请确保目标服务器开启SSH服务，且能正常访问。
